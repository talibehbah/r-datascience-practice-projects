# Data Science Portfolio in R

Practice machine learning projects ready to run in RStudio.

## Quick Start
1. Clone this repository
2. Open the .Rproj file in the project folder to load it in RStudio.
3. Open the .R script to run or edit the code - all data is included!

## Projects
- Adult Income Classification
- Bank Note Authentication
- Bike Share Demand Forecasting
- Boston Housing Price Prediction
- Caravan Insurance Purchase Prediction
- College Admissions Classification
- Employee Attrition Analysis
- Iris Species Classification
- Loan Repayment Prediction
- Student Performance Prediction
- Titanic Survival Prediction
- Wine Quality Clustering

## Requirements
- R 4.0+
- RStudio (recommended)

**Note:** Packages may need to be installed when running the scripts for the first time.
